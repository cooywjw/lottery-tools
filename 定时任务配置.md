# OpenClaw 定时任务配置

## 足球赛事分析自动发布定时任务

### 任务目标
每日自动发布足球赛事分析文章到微信公众号草稿箱。

### 执行时间
- **最佳时间**：每日 13:00（比赛开始前5小时）
- **备用时间**：每日 14:00（比赛开始前4小时）
- **检查时间**：每日 15:00（发布后检查）

### 任务配置

#### 方法一：使用OpenClaw Cron功能
```bash
# 创建每日13:00执行的定时任务
openclaw cron create --name "football-analysis-publish" \
  --schedule "0 13 * * *" \
  --command "python C:/Users/Administrator/.openclaw/workspace/scripts/publish_football_analysis.py" \
  --channel webchat \
  --model glm-5
```

#### 方法二：使用Windows任务计划程序
1. 打开"任务计划程序"
2. 创建基本任务
3. 名称：足球赛事分析发布
4. 触发器：每日 13:00
5. 操作：启动程序
6. 程序：`python.exe`
7. 参数：`C:\Users\Administrator\.openclaw\workspace\scripts\publish_football_analysis.py`
8. 起始于：`C:\Users\Administrator\.openclaw\workspace`

#### 方法三：使用PowerShell脚本
```powershell
# 创建计划任务
$action = New-ScheduledTaskAction -Execute "python.exe" -Argument "C:\Users\Administrator\.openclaw\workspace\scripts\publish_football_analysis.py" -WorkingDirectory "C:\Users\Administrator\.openclaw\workspace"
$trigger = New-ScheduledTaskTrigger -Daily -At "13:00"
Register-ScheduledTask -TaskName "FootballAnalysisPublish" -Action $action -Trigger $trigger -Description "每日发布足球赛事分析文章"
```

### 异常处理

#### 常见问题及解决方案
1. **API调用失败**：
   - 检查网络连接
   - 验证appid和appsecret
   - 等待1分钟后重试

2. **内容生成失败**：
   - 检查模板文件是否存在
   - 验证数据源是否可用
   - 使用备用数据源

3. **发布超时**：
   - 增加超时时间
   - 简化文章内容
   - 分批发布

4. **系统错误**：
   - 检查Python环境
   - 验证依赖库
   - 查看系统日志

#### 告警机制
1. **邮件通知**：发布失败时发送邮件
2. **微信通知**：通过公众号发送通知
3. **日志记录**：详细记录错误信息
4. **自动重试**：失败后自动重试3次

### 备份和恢复

#### 数据备份
```powershell
# 每日备份脚本
$backupDir = "D:\work\backup\football_analysis"
$sourceDir = "C:\Users\Administrator\.openclaw\workspace"
$date = Get-Date -Format "yyyyMMdd"

# 备份模板文件
Copy-Item "$sourceDir\templates\*" "$backupDir\templates_$date\" -Recurse -Force

# 备份脚本文件
Copy-Item "$sourceDir\scripts\*" "$backupDir\scripts_$date\" -Recurse -Force

# 备份配置文件
Copy-Item "$sourceDir\skills\wechat-publisher\config.json" "$backupDir\config_$date.json" -Force
```
