## Kokoro TTS 安装（2026-04-04）

### 安装情况
- Python包：`pip install kokoro-onnx soundfile` ✅ 已安装 (v0.4.7)
- 模型文件：
  - `D:\work\media\kokoro-v1.0.int8.onnx` (88MB) ✅
  - `D:\work\media\voices-v1.0.bin` (27MB) ✅
- 测试结果：英文/中文均可生成，但默认音色包只有英文发音人
- 中文发音质量：能用但有口音（非native中文）

### 使用方法
```python
from kokoro_onnx import Kokoro
import soundfile as sf
kokoro = Kokoro('D:/work/media/kokoro-v1.0.int8.onnx', 'D:/work/media/voices-v1.0.bin')
audio, sr = kokoro.create('文本内容', voice='af_sarah', speed=1.0)
sf.write('output.wav', audio, sr)
```

### 已知问题
- 默认 voices 包只有54个英文voice，无中文voice
- 中文发音有英文口音（可接受）
- 如需更标准的中文发音，需寻找中文voice包
